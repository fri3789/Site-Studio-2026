ZIPを公開する方法 (初心者でも簡単)

1. まずは自分のパソコンで確認

  1. ダウンロードしたZIPファイルを右クリック → 「すべて展開」
  2. 出てきたフォルダを開く
  3. index.html をダブルクリック → ブラウザで表示されます

2. Netlify Drop (最も簡単・無料・アカウント不要)

  1. app.netlify.com/drop にアクセス
  2. 先ほど展開したフォルダを、画面の枠内にドラッグ＆ドロップ
  3. 数秒待つと https://◯◯◯.netlify.app というURLが発行されます
  4. そのURLをブラウザで開けば完成！ SNSでシェアもできます

3. GitHub Pages (無料・GitHubアカウントが必要)

  1. GitHubにログインし、「New repository」で新しいリポジトリを作成 (Public)
  2. リポジトリ名を入力 (例: my-site) → 「Create repository」
  3. 「uploading an existing file」をクリック
  4. 展開したフォルダの中の全てのファイルをドラッグ＆ドロップ → 「Commit changes」
  5. 「Settings」→「Pages」→「Branch」を main / /(root) に設定 → 「Save」
  6. 1分ほど待つと、https://あなたのユーザー名.github.io/リポジトリ名/ で公開されます

4. Cloudflare Pages (無料・高速)

  1. Cloudflareダッシュボードにログイン → 「Pages」→「Create a project」→「Direct upload」
  2. 展開したフォルダをドラッグ＆ドロップ → 「Deploy」
  3. 自動で発行されたURLにアクセス
補足: ZIPに含まれている「.nojekyll」という空ファイルは、GitHub Pagesに「Jekyllという変換ツールを使わないで」と指示するためのものです。他のサービスでは無視されるので気にしなくて大丈夫です。
